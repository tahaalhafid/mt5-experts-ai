COMPACTNESS RULES
- Operator Copilot responses = concise.
- Council advisory packets = compact structured packet.
- Incident reports = medium detail structured.
- Daily operating intelligence reports = medium detail.
- Three-day institutional reviews = broader bounded detail.
- Executive admin briefs = concise executive summary.
- Reviewable proposals = bounded structured detail.
- Learning governance evidence = medium structured detail.

GLOBAL TEMPLATE RULES
- Every template must include a non-authoritative notice.
- Every template must include confidence and evidence framing unless the template is explicitly answer-only; answer-only still requires confidence and limitations.
- Any partial or bounded retrieval must be reflected in limitations.
- Reviewable proposals must explicitly state direct_apply_allowed=false and auto_apply_allowed=false.
- Council advisory packets must explicitly state directional_generation_allowed=false and direct_control_allowed=false.
