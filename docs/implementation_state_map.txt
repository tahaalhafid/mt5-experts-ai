Classification meanings:
- IMPLEMENTED_RUNTIME_ACTIVE        = present and part of the accepted active runtime behavior/path

- DESIGN_PLACEHOLDER_ONLY           = design intent / placeholder exists, but not a completed runtime/governance implementation here
- HISTORICAL_ONLY                   = retained only as historical context, not current operational truth
- NON_AUTHORITATIVE_DERIVED         = derived or placeholder artifact that may be useful operationally but is not authoritative status/control truth
