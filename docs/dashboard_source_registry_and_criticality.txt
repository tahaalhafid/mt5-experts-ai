- TIER_A_PRIMARY_VISIBILITY = curated status sources safe to drive top-level posture and page-primary widgets
- TIER_B_STRUCTURAL_STATUS = structural or summary support sources safe for bounded secondary detail
- TIER_C_CONTEXTUAL_OPTIONAL = contextual help or fallback sources only, never dominant truth
- TIER_D_NEVER_RENDER_DIRECTLY = raw, sensitive, control, or misleading surfaces that must never be rendered directly
