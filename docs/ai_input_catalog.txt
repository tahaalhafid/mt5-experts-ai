CATALOG NOTES
- ALWAYS_VISIBLE_SUMMARIES are the default first read for AI work.
- ON_DEMAND_RETRIEVAL sources require bounded retrieval rules from docs/ai_access_policy.txt.
- SNAPSHOT_ONLY sources expose metadata only, not broad raw plan/state payload reading.
- FORBIDDEN sources remain forbidden regardless of AI authority state in A1.
- A1 defines input discipline only. It does not grant new AI authority or new execution behavior.
